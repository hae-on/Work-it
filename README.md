# Work-it

### 🍇 Work it
아르바이트를 한눈에 볼 수 있는 웹페이지입니다. <br>
기간별로 아르바이트를 작은 상자 형태로 작성하여 볼 수 있고, 하루동안 일한 시간과 총 금액을 계산할 수 있습니다.

### 🍇 Languages
JS (React 사용) / postCSS

### 🍇 소감
두번째 개인 프로젝트. 9월부터 아르바이트를 시작했는데 내가 일한 시간에 따라 월급을 받는 시스템이였다. 내가 기계가 아니기에 매번 시작하는 시간과 끝나는 시간이 달랐다. 딱 떨어지는 수이면 단순하게 계산할 수 있었지만, 애매모호하게 끝났을 경우 알바비를 계산하기 쉽지 않았다. 결국 하루가 꼬이기 시작하면 전체 월급을 알 수 없었다. 조금 더 알바비를 계산하기 쉽게, 내가 하는 일들과 시간에 따라 구분하기 쉽게 만들어보았다. 여러번 엎기도 했고, 레파지토리를 새로 파기도 하는 등 여러 고난과 역경(?)이 있었지만 무사히 끝낼 수 있었다. 데이터베이스를 구현하지 못해 기록을 쌓아두지는 못하지만 알바비를 계산해야 할 때 종종 들어와 사용할 것이다.
